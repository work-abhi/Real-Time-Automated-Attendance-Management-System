# 🧑‍💼 REAL-TIME AUTOMATED ATTENDANCE MANAGEMENT SYSTEM
A smart and secure Attendance Management System developed using Python, Flask, MongoDB, Face Recognition, OpenCV, JWT Authentication, HTML, CSS, JavaScript, and Bootstrap to automate employee attendance with real-time IN/OUT tracking, facial verification, location monitoring, and admin management.

---

# 🚀 Overview

The Real-Time Automated Attendance Management System is an AI-powered web application designed to automate employee attendance management using Face Recognition and real-time tracking technologies. The system allows employees to securely mark attendance through facial authentication while administrators can monitor attendance records, employee activity, and reports from a centralized dashboard.

This project eliminates manual attendance processes and improves accuracy, security, and efficiency in organizations.
